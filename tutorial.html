1XAMPP Setup & Installation

What is XAMPP?

XAMPP is a free, open-source cross-platform web server that includes Apache HTTP Server, MySQL database, and PHP interpreter. It's perfect for local development.

ü§î Why XAMPP?
  • All-in-one package: No need to install Apache, MySQL, and PHP separately
  • Easy setup: One-click installation and configuration
  • Cross-platform: Works on Windows, macOS, and Linux
  • Local development: Test your websites without needing internet

Installation Steps:

  1. Download XAMPP from https://www.apachefriends.org/
  2. Install XAMPP (default installation is fine)
  3. Start XAMPP Control Panel
  4. Start Apache and MySQL services
  5. Verify installation by visiting http://localhost in your browser
Important: Make sure both Apache and MySQL are running (green status) before proceeding.

2Database Creation

Accessing phpMyAdmin

  1. Open your browser and go to http://localhost/phpmyadmin
  2. You should see the phpMyAdmin interface

Creating the Database

  1. Click on "New" in the left sidebar
  2. Enter database name: student_crud
  3. Click "Create"

Creating the Students Table

Click on your database name, then click "SQL" tab and run this query:

CREATE TABLE students (
    id INT(11) NOT NULL AUTO_INCREMENT,
    first_name VARCHAR(50) NOT NULL,
    last_name VARCHAR(50) NOT NULL,
    age INT(3) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    PRIMARY KEY (id)
);
üîç SQL Query Explanation:
  • CREATE TABLE students - Creates a new table named "students"
  • id INT(11) NOT NULL AUTO_INCREMENT - Unique identifier, automatically increases
  • VARCHAR(50) - Variable character field, max 50 characters
  • INT(3) - Integer field, max 3 digits
  • TIMESTAMP DEFAULT CURRENT_TIMESTAMP - Automatically records creation time
  • PRIMARY KEY (id) - Sets id as the unique identifier

4Database Configuration File

Create: db_config/dbconn.php

<?php
    define('DB_HOST', 'localhost');
    define('DB_USER', 'root');
    define('DB_PASS', '');
    define('DB_NAME', 'student_crud');

    $connection = mysqli_connect(DB_HOST, DB_USER, DB_PASS, DB_NAME);

    if(!$connection){
        die("Connection failed: " . mysqli_connect_error());
    }
?>
üîç Code Analysis - Line by Line:
  • <?php - Opens PHP code block
  • define('DB_HOST', 'localhost'); - Creates a constant for database host
  • define('DB_USER', 'root'); - Creates a constant for username (default XAMPP)
  • define('DB_PASS', ''); - Creates a constant for password (empty by default in XAMPP)
  • define('DB_NAME', 'student_crud'); - Creates a constant for our database name
  • mysqli_connect() - Establishes connection to MySQL database
  • if(!$connection) - Checks if connection failed
  • die() - Stops script execution and shows error message
  • mysqli_connect_error() - Gets the last connection error message
ü§î Why Use define() for Constants?
  • Immutable: Constants can't be changed once set
  • Global scope: Available throughout the entire script
  • Security: Less chance of accidental modification
  • Convention: Standard practice for configuration values
ü§î Why Separate Database Configuration?
  • Reusability: Multiple files can include this one connection
  • Security: Database credentials in one secure location
  • Maintainability: Change database settings in one place
  • Environment Management: Easy to switch between development/production databases

7Backend Processing - crud/create_student.php

Complete Code:

<?php
include '../db_config/dbconn.php';

if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    // Get POST data
    $first_name = trim($_POST['first_name']);
    $last_name = trim($_POST['last_name']);
    $age = intval($_POST['age']);

    // Validate input
    if (empty($first_name) || empty($last_name) || $age <= 0) {
        header('Location: ../index.php?error=' . urlencode('Please fill in all fields with valid data'));
        exit;
    }

    // Prepare and execute insert query
    $sql = "INSERT INTO students (first_name, last_name, age) VALUES (?, ?, ?)";
    $stmt = mysqli_prepare($connection, $sql);

    if ($stmt) {
        mysqli_stmt_bind_param($stmt, "ssi", $first_name, $last_name, $age);

        if (mysqli_stmt_execute($stmt)) {
            mysqli_stmt_close($stmt);
            mysqli_close($connection);
            header('Location: ../index.php?success=' . urlencode('Student created successfully!'));
            exit;
        } else {
            mysqli_stmt_close($stmt);
            mysqli_close($connection);
            header('Location: ../index.php?error=' . urlencode('Error creating student: ' . mysqli_error($connection)));
            exit;
        }
    } else {
        mysqli_close($connection);
        header('Location: ../index.php?error=' . urlencode('Error preparing statement: ' . mysqli_error($connection)));
        exit;
    }
} else {
    header('Location: ../index.php?error=' . urlencode('Invalid request method'));
    exit;
}
?>

Step-by-Step Analysis:

1. File Inclusion and Path

Path Resolution:
  • include '../db_config/dbconn.php' - Includes database connection from parent directory
  • ../ - Moves up one directory level (from crud/ to PHP_CRUD/)
  • Why relative path? The file is in crud/ folder, but dbconn.php is in db_config/

2. Request Method Validation

Security Check:
  • $_SERVER['REQUEST_METHOD'] - Gets the HTTP method used to access the page
  • == 'POST' - Ensures only POST requests are processed
  • Why check? Prevents users from accessing this page directly via URL (GET request)

3. Data Retrieval and Sanitization

Input Processing:
  • $_POST['first_name'] - Gets form data from the name attribute
  • trim() - Removes whitespace from beginning and end of string
  • intval() - Converts string to integer, returns 0 if conversion fails
ü§î Why Sanitize Input Data?
  • Security: Prevents malicious input
  • Data Quality: Ensures clean, consistent data
  • Type Safety: Guarantees expected data types
  • User Experience: Handles accidental whitespace

4. Input Validation

Validation Logic:
  • empty($first_name) - Checks if string is empty or contains only whitespace
  • $age <= 0 - Ensures age is a positive number
  • || - Logical OR operator: if ANY condition is true, validation fails

5. Prepared Statements (Security)

Prepared Statement Process:
  • mysqli_prepare() - Prepares SQL statement for execution
  • ? - Placeholders for values to be bound later
  • mysqli_stmt_bind_param() - Binds variables to placeholders
  • "ssi" - Data types: string, string, integer
  • mysqli_stmt_execute() - Executes the prepared statement
ü§î Why Use Prepared Statements?
  • SQL Injection Prevention: User input cannot modify SQL structure
  • Performance: Statement is compiled once, can be executed multiple times
  • Automatic Escaping: Special characters are handled automatically
  • Type Safety: Ensures data types match database expectations

6. Response Handling

Response Process:
  • header('Location: ../index.php?success=...') - Redirects user back to main page
  • urlencode() - Properly encodes message for URL transmission
  • exit - Stops script execution after redirect
  • mysqli_stmt_close() - Frees resources used by prepared statement
  • mysqli_close() - Closes database connection
ü§î Why Redirect Instead of Showing Response?
  • User Experience: User sees the updated data immediately
  • PRG Pattern: Post-Redirect-Get prevents duplicate submissions
  • URL Cleanliness: User doesn't stay on the processing page
  • Browser History: Clean navigation experience

7. Error Handling

Error Management:
  • Validation Errors: User-friendly messages for form validation
  • Database Errors: Technical errors with mysqli_error() details
  • System Errors: Preparation failures and unexpected issues
  • Method Errors: Wrong HTTP method usage

Step 8.1: Database Setup for Authentication

Create Users Table

First, we need to create a users table to store user information. Create a file called db_config/create_users_table.php:

<?php
include 'dbconn.php';

// Create users table for authentication
$sql = "CREATE TABLE IF NOT EXISTS users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
)";

if (mysqli_query($connection, $sql)) {
    echo "Users table created successfully!";
} else {
    echo "Error creating table: " . mysqli_error($connection);
}

mysqli_close($connection);
?>
üîç Database Table Analysis:
  • id INT AUTO_INCREMENT PRIMARY KEY - Unique identifier for each user
  • name VARCHAR(100) NOT NULL - User's full name, required field
  • email VARCHAR(100) UNIQUE NOT NULL - Email address, must be unique and required
  • password VARCHAR(255) NOT NULL - Hashed password, 255 chars for hash storage
  • created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP - Automatically records registration time
ü§î Why These Field Types?
  • VARCHAR(100): Sufficient for names and emails, prevents excessive storage
  • VARCHAR(255): Standard length for password hashes (bcrypt, etc.)
  • UNIQUE email: Prevents duplicate accounts with same email
  • AUTO_INCREMENT: Automatically generates unique IDs

Registration Backend - auth_crud/register_process.php

<?php
include '../../db_config/dbconn.php';

// Add error handling for database connection issues
try {
    if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $name = trim($_POST['name']);
    $email = trim($_POST['email']);
    $password = $_POST['password'];
    $confirm_password = $_POST['confirm_password'];

    // Validate input
    if (empty($name) || empty($email) || empty($password) || empty($confirm_password)) {
        header('Location: ../register.php?error=' . urlencode('Please fill in all fields'));
        exit;
    }

    // Validate email format
    if (!filter_var($email, FILTER_VALIDATE_EMAIL)) {
        header('Location: ../register.php?error=' . urlencode('Please enter a valid email address'));
        exit;
    }

    // Validate password length
    if (strlen($password) < 6) {
        header('Location: ../register.php?error=' . urlencode('Password must be at least 6 characters long'));
        exit;
    }

    // Check if passwords match
    if ($password !== $confirm_password) {
        header('Location: ../register.php?error=' . urlencode('Passwords do not match'));
        exit;
    }

    // Check if email already exists
    $check_sql = "SELECT id FROM users WHERE email = ?";
    $check_stmt = mysqli_prepare($connection, $check_sql);

    if ($check_stmt) {
        mysqli_stmt_bind_param($check_stmt, "s", $email);
        mysqli_stmt_execute($check_stmt);
        $result = mysqli_stmt_get_result($check_stmt);

        if (mysqli_fetch_assoc($result)) {
            mysqli_stmt_close($check_stmt);
            mysqli_close($connection);
            header('Location: ../register.php?error=' . urlencode('Email already exists. Please use a different email.'));
            exit;
        }
        mysqli_stmt_close($check_stmt);
    }

    // Hash password
    $hashed_password = password_hash($password, PASSWORD_DEFAULT);

    // Insert new user
    $sql = "INSERT INTO users (name, email, password) VALUES (?, ?, ?)";
    $stmt = mysqli_prepare($connection, $sql);

    if ($stmt) {
        mysqli_stmt_bind_param($stmt, "sss", $name, $email, $hashed_password);

        if (mysqli_stmt_execute($stmt)) {
            mysqli_stmt_close($stmt);
            mysqli_close($connection);
            header('Location: ../register.php?success=' . urlencode('Registration successful! You can now login.'));
            exit;
        } else {
            mysqli_stmt_close($stmt);
            mysqli_close($connection);
            header('Location: ../register.php?error=' . urlencode('Error creating account: ' . mysqli_error($connection)));
            exit;
        }
    } else {
        mysqli_close($connection);
        header('Location: ../register.php?error=' . urlencode('Database error'));
        exit;
    }
} else {
    header('Location: ../register.php?error=' . urlencode('Invalid request method'));
    exit;
}
} catch (mysqli_sql_exception $e) {
    // Handle MySQL errors
    error_log("MySQL Error: " . $e->getMessage());
    header('Location: ../register.php?error=' . urlencode('Database connection error. Please try again.'));
    exit;
} catch (Exception $e) {
    // Handle other errors
    error_log("General Error: " . $e->getMessage());
    header('Location: ../register.php?error=' . urlencode('An error occurred. Please try again.'));
    exit;
}
?>

  • trim() - Removes whitespace from input data

  • strlen($password) < 6 - Checks minimum password length
  • $password !== $confirm_password - Ensures passwords match
  • SELECT id FROM users WHERE email = ? - Checks for duplicate emails

<?php
session_start();
include '../../db_config/dbconn.php';

if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $email = trim($_POST['email']);
    $password = $_POST['password'];

    // Validate input
    if (empty($email) || empty($password)) {
        header('Location: ../login.php?error=' . urlencode('Please fill in all fields'));
        exit;
    }

    // Check if email exists
    $sql = "SELECT id, name, email, password FROM users WHERE email = ?";
    $stmt = mysqli_prepare($connection, $sql);

    if ($stmt) {
        mysqli_stmt_bind_param($stmt, "s", $email);
        mysqli_stmt_execute($stmt);
        $result = mysqli_stmt_get_result($stmt);

        if ($row = mysqli_fetch_assoc($result)) {
            // Verify password
            if (password_verify($password, $row['password'])) {
                // Login successful - set session variables
                $_SESSION['user_id'] = $row['id'];
                $_SESSION['user_name'] = $row['name'];
                $_SESSION['user_email'] = $row['email'];
                $_SESSION['logged_in'] = true;

                mysqli_stmt_close($stmt);
                mysqli_close($connection);

                // Redirect to index page
                header('Location: ../../index.php?success=' . urlencode('Login successful! Welcome back, ' . $row['name']));
                exit;
            } else {
                mysqli_stmt_close($stmt);
                mysqli_close($connection);
                header('Location: ../login.php?error=' . urlencode('Invalid email or password'));
                exit;
            }
        } else {
            mysqli_stmt_close($stmt);
            mysqli_close($connection);
            header('Location: ../login.php?error=' . urlencode('Invalid email or password'));

Step 8.5: Security Features Explained

Password Hashing

Why Hash Passwords?
  • password_hash($password, PASSWORD_DEFAULT) - Creates secure hash
  • password_verify($password, $hash) - Verifies password against hash
  • One-way encryption: Hash cannot be reversed to original password
  • Salt included: Each hash includes unique salt for security
  • Adaptive: PASSWORD_DEFAULT uses strongest available algorithm

Prepared Statements

SQL Injection Prevention:
  • mysqli_prepare($connection, $sql) - Prepares statement with placeholders
  • mysqli_stmt_bind_param($stmt, "s", $email) - Binds parameters to placeholders
  • mysqli_stmt_execute($stmt) - Executes prepared statement
  • Parameter separation: SQL structure and data are separated
  • Automatic escaping: Special characters are handled safely

Input Validation

Data Validation Methods:
  • trim() - Removes whitespace
  • empty() - Checks for empty values
  • filter_var($email, FILTER_VALIDATE_EMAIL) - Validates email format
  • strlen($password) < 6 - Checks minimum length
  • htmlspecialchars() - Prevents XSS attacks
